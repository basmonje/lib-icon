# Simple biblioteca de iconos de react

Biblioteca de iconos para React de uso personal.

# Instalación

```bash
npm i @basmonje/libicon
```

# Uso

```js
import { Menu } from "@basmonje/libicon"

const Navbar = () => (
    <nav>
        <Menu>
    </nav>
)

```
